# Setup - MAC

* Install `uv`
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
* `python=3.11`
    * `see .python-version this is what is used`
* `make`
```bash
brew install make
```
* ffmpeg
```bash
brew install ffmpeg
```

# Run Example

```bash
make run
```
